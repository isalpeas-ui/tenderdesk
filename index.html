Create a GitHub Personal Access Token
GitHub Gist stores your workspace data online so the whole team shares it in real time — and Claude can read and write it directly.
  1. Go to github.com and sign in (or sign up free)
  2. Go to Settings → Developer settings → Personal access tokens → Tokens (classic)
  3. Click Generate new token, give it a name, tick the gist scope, and generate
  4. Copy the token (starts with ghp_)
